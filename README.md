## Installation

* **Clone the repository**

```
git clone https://github.com/duyanh120032/google-clone.git
```

## Install the dependencies
```bash
cd google-clone
yarn install
```

### To run the app in development mode (hot-code reloading, error reporting, etc.)
```bash
yarn dev
```


### Build the application
```bash
yarn build
```
